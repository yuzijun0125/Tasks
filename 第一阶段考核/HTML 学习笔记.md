# HTML 学习笔记
## 一、HTML 核心认知
- HTML（HyperText Markup Language）即超文本标记语言，是构建网页的骨架，负责定义页面结构和内容（非样式 / 交互）。
* 本质：纯文本标记语言，浏览器解析标签渲染页面，错误标签会被忽略（容错性强）。
* 核心：由「标签、属性、内容」组成，标签分单标签（自闭合）和双标签（成对出现）。
* 标准结构：所有 HTML 文件需遵循固定文档结构，确保浏览器正常解析。
``` html
<!DOCTYPE html> <!-- 声明文档类型为HTML5 -->
<html lang="zh-CN"> <!-- 根标签，lang指定页面语言 -->
<head> <!-- 页面元信息区，不直接显示 -->
  <meta charset="UTF-8"> <!-- 字符编码（必写），避免中文乱码 -->
  <title>我的第一个页面</title> <!-- 浏览器标签页标题 -->
</head>
<body> <!-- 页面可视内容区，所有显示元素放这里 -->
  Hello HTML! <!-- 页面主体内容 -->
</body>
</html>
```
## 二、基础标签分类与用法
### 1. 文本相关标签（语义化优先）
``` html
| 标签 | 功能描述              | 特点           |
| ---------| ----------------- | ------------ |
| <h1> -<h6>| 标题标签（1 级最重，6 级最轻） | 自带加粗、换行，语义明确 |
|   <p> | 段落标签              | 自动换行，上下留间距   |
|   <strong> | 强调文本（重要性）         | 语义化加粗（推荐）    |
|    <em>| 强调文本（语气）          | 语义化斜体（推荐）    |
|  <br>  | 强制换行              | 单标签，无闭合      |
|  <hr>  | 水平分隔线             | 单标签，自带样式分隔   |
```
### 2. 容器与布局标签
``` html
* :<div>块级容器：
* ：<span>行内容器，不换行，用于包裹小段文本（如高亮关键词）。
* 核心用途：结合 CSS 实现布局，是页面结构的 “积木”。
```
### 3. 媒体标签
* 图片标签 （单标签）：
``` html
 <img src="图片路径" alt="图片加载失败提示" width="200" height="150">
```
* src：必选属性，支持本地路径（相对 / 绝对）或网络 URL。
* alt：无障碍必备，屏幕阅读器读取，图片失效时显示。
- 音频标签 ：您的浏览器不支持音频播放&#x20;
``` html
<audio src="音频路径" controls autoplay loop>您的浏览器不支持音频播放</audio>
```
* controls：显示播放控件，autoplay 自动播放（部分浏览器限制），loop 循环。
- 视频标签 ：
``` html
<video src="视频路径" controls width="400" poster="封面图路径"></video>
```
* poster：视频加载前显示的封面图。
### 4. 链接标签 
``` html
<!-- 1. 跳转到外部网页 -->
<a href="https://www.baidu.com" target="_blank">打开百度（新窗口）</a>
<!-- 2. 跳转到本地页面 -->
<a href="about.html">关于我们</a>
<!-- 3. 锚点跳转（同一页面内） -->
<a href="#section1">跳转到第一部分</a>
<div id="section1">第一部分内容</div>
<!-- 4. 下载文件 -->
<a href="资料.pdf" download>下载PDF资料</a>
```
- href：目标地址（必填），target="_blank" 新窗口打开（默认当前窗口）。
- 锚点跳转：需给目标元素设置 id，href 对应 #id。
## 三、列表与表格（结构化数据）
### 1. 列表标签
* 无序列表 （默认圆点标记）：
``` html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
</ul>
```
* 有序列表 （默认数字标记）：
``` html
<ol start="3"> <!-- 从3开始计数 -->
  <li>第一步</li>
  <li>第二步</li>
</ol>
```
* 自定义列表 （术语 - 解释结构）：
``` html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，构建页面结构</dd>
</dl>
```
* 注意： 是列表项唯一子标签，不可嵌套其他标签。
### 2. 表格标签（展示二维数据）
``` html
<table border="1"> <!-- border设置边框（默认无） -->
  <thead> <!-- 表头区 -->
    <tr> <!-- 行 -->
      <<th>姓名</</th> <!-- 表头单元格（加粗居中） -->
      <<th>年龄</</th>
    </tr>
  </thead>
  <tbody> <!-- 表体区 -->
    <tr>
      <td>张三</td> <!-- 普通单元格 -->
      <td>20</td>
    </tr>
    <tr>
      <td colspan="2">合计：1人</td> <!-- colspan跨列合并 -->
    </tr>
  </tbody>
</table>
```
- 核心属性：colspan（跨列）、rowspan（跨行）、border（边框）。
- 语义化结构：<thead>+<tbody>+<<tfoot>，提升可读性和 SEO。
## 四、表单标签（用户交互核心）
用于收集用户输入（如登录、注册、搜索），需配合  容器使用。

### 1. 表单容器
``` html
<form action="提交地址" method="get/post">
  <!-- 表单元素 -->
</form>
action：表单数据提交的后端地址（如 PHP 文件）。
method：提交方式（get 明文传参，适用于搜索；post 隐式传参，适用于登录 / 注册）。
```

### 2. 常用表单元素
| 元素类型  | 代码示例 | 功能描述           |
| -----    | ----     | --------------    |
| 文本输入框 |  <input type="text" name="username" placeholder="请输入用户名"> | 输入单行文本         |
| 密码输入框 |    <input type="password" name="pwd">  | 输入密码（隐藏显示）     |
| 单选按钮  |  <input type="radio" name="gender" value="male" checked> 男   | 单选（name 相同为一组） |
| 复选框   |  <input type="checkbox" name="hobby" value="reading"> 阅读 | 多选             |
| 下拉菜单  |  <select name="city"><option value="sh">上海</option></select>| 下拉选择           |
| 文本域   | <textarea name="intro" rows="5" cols="30">个人简介</textarea> | 多行文本输入         |
| 提交按钮  |     <input type="submit" value="登录"> | 提交表单数据         |
| 重置按钮  |    <input type="reset" value="重置">  | 清空表单输入         |

* 关键属性：name（表单数据的键名，后端接收用）、value（提交的实际值）、placeholder（输入提示）、checked（默认选中）。

## 五、HTML5 新增特性（重点）
### 1. 语义化布局标签（替代，提升可读性）
``` html
<header>：页面头部（导航、Logo）。
<nav>：导航栏（菜单链接）。
<main>：页面主体内容（唯一）。
<section>：内容区块（如文章章节）。
<article>：独立文章（如博客、新闻）。
<footer>：页面底部（版权、联系方式）。
  ```
### 2. 新增表单类型
* type="email"：邮箱验证（自动检测格式）。

* type="tel"：电话输入（移动端弹出数字键盘）。

* type="date"：日期选择器（无需手动输入）。

* type="number"：数字输入（限制正负、步长）。

* type="search"：搜索框（自带清除按钮）。
### 3. 其他实用特性
``` html
<canvas>：画布，用于绘制图形、动画（需配合 JavaScript）。
<svg>：矢量图形（放大不失真，如图标）。
```
localStorage：本地存储（持久化保存数据，无过期时间）。
## 六、学习重点与避坑指南
### 1. 核心原则
* 语义化优先：优先使用有含义的标签（如而非），提升 SEO 和可维护性。
* 标签嵌套规范：块级标签可包含行内 / 块级标签（如包含），行内标签不可包含块级标签（如不可包含）。
* 闭合标签：双标签必须成对闭合，单标签建议加 /（如），兼容老浏览器。
### 2. 常见错误
* 忘记写，导致中文乱码。

* 表单元素未加 name 属性，后端无法接收数据。

* 图片路径错误（相对路径需注意文件层级，如../images/1.jpg 表示上一级目录）。

* 嵌套混乱（如内直接写文本，未用包裹）。
