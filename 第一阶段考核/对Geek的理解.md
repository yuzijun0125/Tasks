# *对Geek的理解*

-Geek精神的核心***是对未知的极致好奇、对专业的偏执追求，以及用技术解决问题的实践主义。***它并非“沉迷小众”的标签，而是一种兼具探索欲与行动力的态度。Geek们不满足于“知其然”，更执着于“知其所以然”，会为弄清事物原理主动拆解、钻研；在专注领域追求极致，拒绝“差不多”，苛求细节与效率；遇到问题不空谈，习惯用知识或创意落地解决方案，比如编写工具简化流程、改造设备满足需求。同时，他们秉持开放共享的态度，乐于交流经验、贡献成果，让知识流动创造更大价值。这种探索、较真、务实与共享，正是Geek精神的核心底色。

# *对Git Hub的理解*

## 个人理解

-Git Hub 就是个「代码界的网盘+朋友圈+协作平台」三合一。

网盘功能（存代码）：你把报告存在 Git Hub 上，再也不怕电脑死机文件丢失了。它还能帮你记下每次修改的内容，随时可以找回之前的版本。

朋友圈功能（看代码）：你可以把报告公开，让所有人都能看到。别人觉得你写得好，可以给你点赞（Star）或者收藏（Fork）一份回去学习参考。

协作功能（改代码）：
所以，Git Hub 就是程序员用来存代码、分享代码、和全世界的人一起合作写代码的地方。 我们现在用的手机App、网站软件，绝大部分背后都有它的功劳。

## 对其中一些关键词的理解

-**Fork**：复制，将想修改的代码拷贝到自己的副本中。

-**Pull request**:如果团队一起完成一个项目，我想对方案进行修改，可以按下***Fork***,修改无误后，可以发起***Pull request***。

# *对于进不去网页的问题*

-I P 被墙：Git Hub 的某些服务器 IP 地址可能受到限制，导致连接中断或速度极慢。
-浏览器问题：浏览器缓存、Cookie 或扩展程序（插件）干扰。

-可以通过加速器解决
