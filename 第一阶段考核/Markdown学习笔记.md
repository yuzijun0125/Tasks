# Markdown学习笔记
## 为什么要用Markdown
- 简单：语法直观，易于学习和手写
- 通用：GitHub、知乎、Notion 等众多平台都支持。
- 高效：专注于内容而非排版，纯键盘即可完成写作。
- 易转换：可以轻松转换为 HTML、PDF 等格式。
## 基础语法
### 1. 标题
- 再首行用#来表示标题，数量表示级别。
### 2. 粗体和斜体
- 用#和*包裹文体
### 3. 列表
- 无序列表使用-，+，或者空格
- 有序列表使用.加空格
### 4.链接加图片
- [链接文字](链接地址)
- [访问Git Hub](https://github.com)
### 5.代码
- 行内代码
  - 用一个反引号`包裹
- 代码块
  -用三个反引号```
  ``` c
  #inclide <stdio.h>
  int main ()
  {
   printf("hello");
   return 0;
  }
  ```
### 6.引用
- 在段前行首使用》
### 7.分割线
- 三个或者三个以上的***,---
## 常用扩展语法
### 1. 表格
- 使用|来分割不同单元格，--来分割表头和其他行
### 2.任务列表
- 在无序列表中只用[ ],[x]
- [ ] 待办事项
- [x] 已完成
### 3.删除线
用两个~~包裹文本
