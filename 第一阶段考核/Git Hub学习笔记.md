# Git Hub学习笔记
## 什么是Git Hub
GitHub 一个基于 Git 的代码托管平台（网站）。它提供了一个中心服务器，让多人可以协作开发同一个项目。 就像代码的“云盘”或“朋友圈”。你可以把本地存档（项目）上传到云端备份，也可以和朋友一起协作。
## 对于该网站的理解
### 注册账号
打开 GitHub网站，首页如下图所示，点击右上角Sign up按钮，进入注册页面。使用电子邮件注册，设置好用户名和密码，即可生成账号。完成注册后，点击旁边的Sign in按钮登录。 
### 设置个人资料
在任何页面的右上角，选择 ，然后单击“新建存储库”。
在“Repository name（仓库名称）”下，输入与您的 GitHub ***用户名匹配***的仓库名称。 
选择“公共”。
选择“使用 README 初始化此存储库”。
### 设计READMEmd文件
运用Mardown语句，对个人封面进行修改
### 建立Tasks仓库
在“存储库名称”框中，键入 hello-world。

在“说明”框中，键入简短说明。例如，键入“此存储库用于练习 GitHub 流”。

选择存储库是“公共”还是“专用” 。

选择“添加 README 文件”。

单击“创建存储库”。
